# 🗳️ Soroban Voting App

## 📌 Project Description

This project is a basic decentralized voting application built using the Soroban smart contract platform on Stellar. It demonstrates how to create, manage, and interact with a simple on-chain voting system where users can vote for predefined candidates.

The goal of this project is to showcase fundamental smart contract concepts such as state storage, function invocation, and data handling using Soroban SDK.

---

## ⚙️ What It Does

* Initializes a list of candidates on deployment
* Allows users to vote for a candidate
* Tracks vote counts securely on-chain
* Provides functions to retrieve individual vote counts
* Displays all voting results

All votes are stored within the contract’s persistent storage, ensuring transparency and immutability.

---

## ✨ Features

### 🧩 Core Features

* **Candidate Initialization**

  * Set up a list of candidates at contract deployment

* **Voting Mechanism**

  * Users can cast votes for any available candidate

* **Vote Tracking**

  * Each vote increments a candidate's total count

* **Result Retrieval**

  * Fetch vote count for a specific candidate
  * Retrieve all candidates with their vote totals

---

### 🔐 Blockchain Benefits

* **Transparency** — All votes are publicly verifiable
* **Immutability** — Votes cannot be altered once recorded
* **Decentralization** — No central authority controls the voting

---
